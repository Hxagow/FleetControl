# FleetControl


## Instalation

Virtual env :
```bash
python -m venv venv
source venv/bin/activate
```

Dependances :
```bash
pip install -r requirements.txt
cd src/frontend
npm install
```

## Dev.

Django :
```bash
cd src
python manage.py runserver
```

Tailwind :
```bash
cd src/frontend
npm run dev
```
